# ValueLog 사용자 가이드

## 👋 환영합니다!

ValueLog는 아이의 성장 경험을 체계적으로 기록하고 포트폴리오로 전환하는 웹 애플리케이션입니다.

---

## 🚀 시작하기

### 1. 회원가입 및 로그인
1. 앱 접속: `https://your-valuelog-url.vercel.app`
2. **회원가입** 버튼 클릭
3. 이메일과 비밀번호 입력
4. 이메일 인증 완료 (Supabase에서 발송)

### 2. 첫 아이 등록
1. 로그인 후 **"아이 추가"** 버튼 클릭
2. 아이 이름 및 생년월일 입력
3. 저장

---

## 📝 경험 기록하기

### STARR 회고 템플릿
ValueLog는 **STARR 프레임워크**를 사용합니다:

- **S (Situation)**: 어떤 상황이었나요?
- **T (Task)**: 어떤 목표나 과제가 있었나요?
- **A (Action)**: 어떤 행동을 했나요?
- **R (Result)**: 어떤 결과가 있었나요?
- **R2 (Reflection)**: 무엇을 배웠나요?

### 경험 추가 방법
1. **"새 경험 기록"** 버튼 클릭
2. **기본 정보** 입력:
   - 제목 (예: "시립미술관 전시 관람")
   - 날짜
   - 활동 유형 (예: 체험학습)
   - 장소 (선택)
   - 만족도 (1~5점)
   - 대표 이미지 (선택)
3. **역량 및 활동 태그** 선택:
   - 역량: 창의성, 협동심, 끈기 등
   - 활동 분야: 예술, 과학, 스포츠 등
4. **STARR 회고** 작성:
   - 각 질문에 구체적으로 답변
   - 최소 20자 이상 권장
5. **"기록 완료하기"** 클릭

## 📊 통계 및 인사이트

### 대시보드 기능
- **월별 활동 추이**: 시간에 따른 활동량 확인
- **역량 레이더 차트**: 어떤 역량이 많이 발휘되었는지 시각화
- **분야별 만족도**: 어떤 활동에서 만족도가 높았는지 분석
- **AI 인사이트**: 자동 생성된 성장 분석 문구

## 📄 포트폴리오 텍스트 활용

### 자동 생성 텍스트
각 경험 상세 페이지에서 두 가지 텍스트가 자동 생성됩니다:

#### 1. 포트폴리오·일기용 서술형
- STARR 답변을 자연스러운 문단으로 결합
- 일기나 활동 보고서에 활용

#### 2. 요약 텍스트 (자소서·체험학습 보고서용)
- 핵심 역량과 성과를 압축한 1~2문장
- 체험학습 보고서, 자기소개서, 활동 소감문에 바로 활용 가능
- 예시: "시립미술관 어린이 전시 활동에서 작품을 직접 관찰하고 색의 의미를 토론하며 창의성과 감수성 역량을 집중적으로 키운 경험임."

### 복사 방법
- 각 텍스트 박스 옆 **"복사"** 버튼 클릭
- 원하는 문서에 붙여넣기

## ✏️ 경험 수정 및 삭제

### 수정하기
1. 경험 상세 페이지에서 **"내용 수정하기"** 클릭
2. 원하는 항목 수정
3. **"수정 완료하기"** 클릭

### 삭제하기
1. 경험 상세 페이지에서 **"삭제"** 버튼 클릭
2. 확인 메시지에서 **"확인"** 클릭
3. ⚠️ 삭제된 기록은 복구할 수 없습니다

## 💡 활용 팁

### 1. 정기적으로 기록하기
- 활동 직후 바로 기록하면 더 생생한 내용 작성 가능
- 주 1회 정기적으로 기록하는 습관 추천

### 2. 사진 활용
- 활동 사진을 첨부하면 나중에 기억하기 쉬움
- 포트폴리오 제작 시 시각적 자료로 활용 가능

### 3. 구체적으로 작성
- "재미있었다" 대신 "어떤 점이 재미있었는지" 구체적으로 작성
- 숫자나 구체적인 사례 포함 (예: "3명의 친구와 함께...")

### 4. 태그 활용
- 역량 태그를 일관성 있게 선택하면 통계 분석이 더 정확해짐
- 커스텀 태그로 우리 아이만의 특별한 역량 추가 가능

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 여러 아이를 등록할 수 있나요?
**A.** 네, 한 계정에 여러 아이를 등록하고 각각 별도로 관리할 수 있습니다.

### Q2. 데이터는 안전한가요?
**A.** Supabase의 보안 인증 시스템을 사용하며, 각 사용자의 데이터는 완전히 분리되어 저장됩니다.

### Q3. 오프라인에서도 사용할 수 있나요?
**A.** 현재는 인터넷 연결이 필요합니다. PWA로 설치하면 일부 캐싱 기능이 제공됩니다.

### Q4. 이미지가 업로드되지 않아요
**A.** Supabase Storage 버킷 설정을 확인해주세요. 관리자에게 문의하시면 도움을 드립니다.

### Q5. 비밀번호를 잊어버렸어요
**A.** 로그인 페이지에서 **"비밀번호 찾기"** 기능을 사용하세요. (Supabase 기본 제공)
